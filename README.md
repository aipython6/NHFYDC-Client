# vite-vue3-admin

> 基于 vite3.x 重构整个前后端项目，完善后端权限控制细粒度，封装更多场景化组件...正在完善中

基于 vite3.x + vue3.x + antd-design-vue3.x + typescript4.x 的后台管理系统模板

## 安装使用

- 获取项目代码

```bash
git clone https://github.com/buqiyuan/vite-vue3-admin
```

- 安装依赖

```bash
cd vite-vue3-admin

pnpm install

```

- 运行

```bash
pnpm serve
```

- 打包

```bash
pnpm build
```
